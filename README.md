# Avvocato — Legal SaaS QA Project

## 📌 Overview
QA documentation and testing for Avvocato — an AI-powered legal 
SaaS platform for case management built for lawyers.

🔗 Live App: https://avvocato-sooty.vercel.app

## 🛠️ Tools Used
- Postman & API Dog — API Testing
- Google Sheets/Excel — Test Case Management
- Manual Testing — Functional & UI

## ✅ What's Covered
- SRS — Software Requirements Specification
- IQS — Integration Quality Standards  
- Manual test cases — functional, UI, regression
- API testing — endpoints, auth, response validation

## 📂 Structure
├── docs/             → SRS & IQS documents
├── test-cases/       → Manual test case sheets
├── api-testing/      → Postman collections & API Dog exports
└── bug-reports/      → Documented defects
